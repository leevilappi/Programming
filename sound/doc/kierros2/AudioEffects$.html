  14. def fade(sound: StereoSound, fadeLen: Int): StereoSound

    Method takes a sound as a parameter.

    Method takes a sound as a parameter. From the beginning of the sound until fadeLen, sound is faded in (fading factor increases from 0 to 1 in this period). And from the fadeLen samples from the end of the sound to the end of the sound, sound is faded out (fading factor decreases from 1 to 0 in this pediod).

    The method returns a new StereoSound object that is created from modified sound data.

    sound

    The StereoSound object which data is used.

    fadeLen

    The length of fade in the beginning and the end of the sound in number of samples. When sample rate is 44 100, one second is 44 100 samples.

    returns

    New StereoSound object with fade in and fade out effect.

  15. def fadeIn(sound: StereoSound, fadeLen: Int): StereoSound

    Method takes a sound as a parameter.

    Method takes a sound as a parameter. From the beginning of the sound until fadeLen, the sound is faded by multiplying values in left and right channels by fading factor.

    The fading factor gets values between 0 and 1 in equal increments for each sound value over the duration of fadeLen.

    The method returns a new StereoSound object that is created from modified sound data.

    sound

    The StereoSound object which data is used.

    fadeLen

    The length of fade in the beginning of the sound in number of samples. When sample rate is 44 100, one second is 44 100 samples.

    returns

    New StereoSound object with fade in effect.

  16. def fadeOut(sound: StereoSound, fadeLen: Int): StereoSound

  25. def panLeftToRight(sound: StereoSound, panLen: Int): StereoSound

    Method takes a sound as a parameter.

    Method takes a sound as a parameter. From the beginning of the sound until panLen, sound is panned from left to right. This means that left channel is faded out and right channel is faded in simultaneously.

    The method returns a new StereoSound object that is created from modified sound data.

    sound

    The StereoSound object which data is used.

    panLen

    The length in number of samples (from the beginning of a sound) during what time pan effect is produced. When sample rate is 44 100, one second is 44 100 samples.

    returns

    New StereoSound object with pan effect.

  26. def removeVocals(sound: StereoSound): StereoSound

    Method takes a sound as a parameter.

    Method takes a sound as a parameter. Vocals are removed by substracting the right channel's values from the left channel's values and the resulting value is divided by 2 to maintain the volume level.

    The method returns a new StereoSound object that is created from modified sound data.

    sound

    The StereoSound object which data is used to produce a sound with removed vocals

    returns

    New StereoSound object with vocals removed.
